プライバシーポリシー

最終更新日：2026年4月28日

本ポリシーは、Chrome 拡張機能「週末・祝日通知」（以下「本拡張機能」）における情報の取り扱いについて説明します。

1. 収集する情報

本拡張機能は、個人を特定できる情報を収集・送信しません。

端末上のローカルストレージ（chrome.storage.local）に以下の情報のみを保存します：

  • ユーザーが設定したカスタム通知メッセージ（タイトル・本文）
  • 通知を最後に表示した日付（重複通知の防止のため）

これらのデータは端末外部に送信されることはありません。

2. Google カレンダー API の利用

本拡張機能は、日本の祝日情報を取得するために Google カレンダー API（読み取り専用）を使用します。

  • アクセスするデータ：Google が公開している日本の祝日カレンダーのイベント情報のみ
  • スコープ：calendar.events.readonly（読み取り専用）
  • ユーザーの個人カレンダーには一切アクセスしません

Google の OAuth 認証は Chrome の標準機能（chrome.identity）を通じて行われます。認証トークンは本拡張機能が保存・管理するものではなく、Chrome によって安全に管理されます。

3. 第三者への情報提供

本拡張機能は、いかなる第三者に対してもユーザーの情報を販売・提供・開示しません。

4. データの保持と削除

ローカルに保存されたデータは、ユーザーがいつでも削除できます。拡張機能をアンインストールすることで、すべてのローカルデータが削除されます。

5. 本ポリシーの変更

本ポリシーを変更した場合、このページを更新します。重要な変更がある場合は拡張機能のアップデートを通じてお知らせします。

6. お問い合わせ

本ポリシーに関するご質問は、Chrome Web Store の拡張機能ページよりお問い合わせください。
